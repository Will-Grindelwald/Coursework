DESCRIPTION

This file defined VertexBase class to load user Vertex class. VertexBase is an abstract class with virtual methods. Node class is pointed to inside, which implements inner details for corresponding Vertex.

There is a 3-layer inheritance hierarchy about Vertex: VertexBase is on top, Vertex class of template is in the middle, and user Vertex class is at the bottom.
