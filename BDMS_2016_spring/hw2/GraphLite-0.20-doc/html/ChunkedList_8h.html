DESCRIPTION

This file defined ChunkedList class serving as pointer array to optimize visit to messages and better to prefetch.

If there is no freespace on ChunkedList, we allocate a block of m_mem_pool_capacity size once from memory, so as to reduce request times for memory.

Also defined ChunkedList::Iterator class, so as to provide interface for users to visit ChunkedList.
