DESCRIPTION

This file defined Graph class as an interface for users to config arguments and control system runtime to some degree.

System needs to load user files dynamically. That is, we can't know details about subclass implemented by users before whole program linked up. So by means of C++ language features, some pointers of abstract classes are necessary.
